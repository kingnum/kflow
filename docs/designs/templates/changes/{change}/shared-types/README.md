## 说明

本目录存放前后端子变更共同消费的类型定义（DTO / interface / enum），由变更级 design 阶段产出。

**创建条件**：design 阶段完成接口设计后，若存在前后端子变更共同消费的类型定义，则创建此目录。

**跳过条件**：变更不存在跨子变更共享的类型定义（纯后端或前后端无共同 DTO），则不创建此目录。
