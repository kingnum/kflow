## 六、硬约束

### 多文件输出

- 输出到 `docs/changes/{change}/prototype/` 目录
- `index.html` 为访问入口
- 允许 `page-*.html` 独立页面文件
- 允许 `shared/` 共享资源子目录（styles.css、components.js、app-phone.js）

### 业务流程驱动（flow demo 模式）

- 使用 **flow demo 模式**（可交互业务流程）
- 单台设备状态管理器（如 `AppPhone`）驱动页面切换和交互
- 用户可通过点击 tab bar / 按钮 / 标注点完成完整业务流程
- **SHALL NOT** 提供 overview 多屏并排静态展示

### 离线自包含

- 所有 CSS/JS/字体/图片资源**必须内联或相对路径引用**
- **禁止** `http://` `https://` 外部 CDN 依赖
- 字体使用系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
