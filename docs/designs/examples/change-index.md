## 索引说明

### 文件用途

此索引文件用于：
1. 快速查看所有活跃变更的状态
2. 记录已归档变更的信息
3. 支持 `kflow-status` Skill 的索引读取功能
4. 支持跨变更冲突检测（影响文件字段）

### 更新时机

索引文件在以下时机更新：
- 创建新变更时：添加到活跃变更列表，填写影响文件字段
- 归档变更时：从活跃列表移除，添加到已归档列表
- 阶段变更时：更新当前阶段字段
- 影响文件变更时：更新影响文件字段

### 状态总结 Skill 使用

`kflow-status` Skill 在执行时会：
1. 尝试读取此索引文件
2. 从索引获取活跃变更基本信息
3. 读取各变更的 `.status.md` 获取详细状态
4. 输出综合状态汇总表格（含子变更进度矩阵）
